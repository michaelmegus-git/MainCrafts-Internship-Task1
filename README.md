# 📊 Student Performance Analysis

This project was completed as **Task 1** of the **MainCrafts Internship**. It focuses on analyzing student performance data using Python and performing Exploratory Data Analysis (EDA) to uncover patterns, relationships, and insights.

## 📌 Project Objectives

- Load and explore the student dataset
- Clean and preprocess the data
- Perform Exploratory Data Analysis (EDA)
- Visualize important trends and relationships
- Draw meaningful conclusions from the analysis

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/michaelmegus-git/MainCrafts-Internship-Task1.git
```

2. Navigate to the project folder

```bash
cd MainCrafts-Internship-Task1
```

3. Install the required libraries

```bash
pip install pandas numpy matplotlib seaborn jupyter
```

4. Open the notebook

```bash
jupyter notebook student_analysis.ipynb
```
